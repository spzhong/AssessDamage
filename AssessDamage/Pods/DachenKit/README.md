## Installation

DachenKit is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DachenKit', :git => 'http://192.168.3.200/ios/DachenKit.git', :tag => '0.1.0'
```
